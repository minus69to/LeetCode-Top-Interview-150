# LeetCode Top Interview 150

My C++ solutions to the [LeetCode Top Interview 150](https://leetcode.com/studyplan/top-interview-150/) study plan. The problems are organized by topic, following the official LeetCode problem set order. This is an ongoing solve — topics get updated as I work through them.

## Language

All solutions are written in **C++**.

## Topics

| Folder | Topic | What it covers |
|--------|-------|----------------|
| `Array - String` | Array / String | In-place operations, rotations, majority element, string manipulation |
| `Two Pointers` | Two Pointers | Opposite-end and slow/fast pointer patterns |
| `Sliding Window` | Sliding Window | Fixed and variable window over arrays/strings |
| `Matrix` | Matrix | 2D traversal, rotation, spiral order |
| `Hashmap` | Hashmap | Frequency counting, grouping, lookup optimization |
| `Intervals` | Intervals | Merge, insert, and overlap detection |
| `Stack` | Stack | Monotonic stack, expression evaluation |
| `Linked List` | Linked List | Reversal, cycle detection, merge, reorder |
| `Binary Search` | Binary Search | Sorted array search, boundary finding |
| `Binary Search Tree` | BST | Validation, LCA, kth smallest |
| `Binary Tree General` | Binary Tree | DFS traversals, construction, path problems |
| `Binary Tree BFS` | Binary Tree BFS | Level-order traversal, right side view |
| `Graph General` | Graph | DFS, connected components, topological sort |
| `Graph BFS` | Graph BFS | Shortest path, multi-source BFS |
| `Trie` | Trie | Prefix tree insert/search, word dictionary |
| `Heap` | Heap / Priority Queue | Top K elements, median finding |
| `Backtracking` | Backtracking | Combinations, permutations, subsets |
| `Divide & Conquer` | Divide & Conquer | Merge sort based problems |
| `Kadane's Algorithm` | Kadane's | Maximum subarray, max product |
| `1D DP` | 1D Dynamic Programming | Climbing stairs, house robber, coin change |
| `Multidimensional DP` | Multi-dim DP | Edit distance, LCS, knapsack variants |
